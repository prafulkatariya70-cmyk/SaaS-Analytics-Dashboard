# SaaS Business Analytics Dashboard

This project presents an end-to-end SaaS Business Analytics Dashboard built using Power BI to analyze key business metrics such as revenue, customer growth, churn rate, and user engagement.

It uses multiple datasets (customers, revenue, usage) with proper data modeling and relationships.

Key features include:

* KPI cards (Revenue, Customers, Churn Rate, Usage)
* Interactive slicers for dynamic filtering
* Drillthrough functionality for detailed analysis at granular levels
* Custom tooltips for enhanced data exploration and user experience
* Clean and visually appealing dashboard design

The dashboard highlights important business insights such as higher revenue from premium users and higher churn in lower-tier plans.

This project demonstrates strong skills in Power BI, DAX, data modeling, and business-oriented data analysis.
